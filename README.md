  ## 💻 개발 가치관
  ---
  ### * 개발과 운영의 벽을 허물고 협업을 통한 효율화를 지향합니다.
  ### * 투명한 커뮤니케이션과 실시간 피드백을 수반하여 개발합니다.
  ### * Agile 개발을 통해 짧은 주기의 개발과 피드백 로프를 이룹니다.
  ### * 문제가 발생하면 시각화하고 팀 내에서 솔루션을 함께 모색합니다.
  ### * 도메인 주도 개발을 통해 해결해야 할 문제의 영역을 분할해서 개발합니다.
  ### * 백로그를 작성하여 실시간으로 다른 팀원들을 follow-up합니다.

  ## ✊ Strong Point
  ---
  ### 고민보단 Go라는 철학을 바탕으로, 빠르게 배우고 빠르게 시도해보는 실행 중심의 개발자입니다.
  ### 문제가 발생했을 때, 단순히 검색에 의존하기 보다, 관련 커뮤니티 협업에 종사하시는 분들의 의견을 적극적으로 참고하여 솔루션을 다각도로 탐색합니다.
  ### 기존 레거시에 대해 무비판적으로 수용하지 않고, 비판적으로 바라보는 습관을 가지고 있습니다.
  ### 질문을 두려워하지 않습니다.
